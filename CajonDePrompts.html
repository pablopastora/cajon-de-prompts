Configurar Ollama + Llama
Ollama te permite usar modelos de IA de forma completamente gratuita y local, sin enviar nada a internet. Aquí te explicamos cómo instalarlo:
1. Instalar Ollama
→ Visita ollama.com y descarga para tu sistema (Windows/Mac/Linux).

2. Descargar el modelo Llama
→ Abre una terminal y ejecuta:
ollama pull llama3.2

3. Ejecutar Ollama con CORS
→ En la terminal ejecuta:
OLLAMA_ORIGINS=* ollama serve
(en Windows: usa las variables de entorno del sistema)

4. Volver aquí y probar la conexión
